# buddha.py
Small discord bot that returns quotes attributed to Buddha.

## Prerequisistes:
- Python 3.13+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Set-up:
1. Clone respository: `git clone https://github.com/SandorTeleki/buddha.py.git`
2. Move to cloned respository with `cd buddha.py`
3. Install dependencies with `uv sync`
4. Run game with: `uv run python main.py`
